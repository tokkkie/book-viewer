# Book Viewer

Go + Wails + React + TypeScript で実装した書籍ビューワです

世の中の画像ビューワやブックビューワを見ると欲しいものがなかったり、不要な機能がついている事も多く、設定も面倒でした。
このソフトウェアはローカルディレクトリの通りに表示を行い、見開き表示と拡大機能だけを提供するシンプルなビューワです。

将来別の機能が欲しくなるかもしれませんが、おそらくこれで十分です。

## 機能

- **シリーズパネル**: シリーズ一覧を表示
- **ボリュームパネル**: 選択したシリーズの巻を表示
- **ビューワパネル**: 画像を見開き表示（右開き対応）
- **対応形式**: ZIP (.zip, .cbz), RAR (.rar, .cbr), 7z (.7z, .cb7), ディレクトリ
- **ネスト対応**: 圧縮ファイル内のディレクトリ構造を認識し、階層的に表示
- **拡大機能**: ホイールで拡大・縮小、ドラッグで移動（拡大率はページ間で保持）
- **キーボード操作**: 矢印キーでページ送り、Escでビューワを閉じる
- **設定**: ルートディレクトリの選択と永続化
- **状態保持**: パネル幅、ルートディレクトリをlocalStorageに保存

## インストール

### ビルド済みバイナリ

Releasesページからダウンロードしてください。

### ソースからビルド

[Wails](https://wails.io/) のインストールが必要です。

```bash
wails build
```

## 使い方

1. 起動後、SERIESパネルの ⚙️ アイコンからルートディレクトリを選択
2. SERIESでWクリック → VOLUMESをダブルクリックで開く
3. パネル幅はドラッグで調整可能（設定は自動保存）
4. ビューワ内で矢印キー（↓/→）で次ページ、（↑/←）で前ページ
5. ホイールで拡大・縮小、拡大時はドラッグで移動

## VOLUMEファイルの配置

ファイルは以下の構造で配置してください:

```text
/path/to/books/
├── SERIES-DIR/
│   ├── 01/
│   │   ├── 001.jpg
│   │   ├── 002.jpg
│   │   └── ...
│   ├── 02.zip
│   └── ...
└── ...
```

圧縮ファイル内にディレクトリがある場合、各ディレクトリをVOLUMEとして認識します。
